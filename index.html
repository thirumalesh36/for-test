if statement in C

if statement is a decision making statement which executes a block of code only if the given condition is true. If the condition is false then it skips the if block.

In simple words, the if statement first checks the conditon if the condition is true then it executes the code inside if block otherwise it skips the if block.

Ternary operator (?) also did like this type of process. We have already seen this in the C Operators tutorial.

Syntax

 
    if(condition){
        //true block code or instructions
    }

Example

    
 #include<stdio.h>
 int main(){
    int a = 3;
    int b = 9;
    if(a < b){
    printf("Hello welcome to Matrix Codez");
    }
    return 0;
 }
Output

Hello welcome to Matrix Codez

Here the condition is a is less than b so 3 is less than 9 so the given condition is true. We want to print Hello welcome to Matrix Codez if the condition is true so it happened.

Let's see another example

    
 #include<stdio.h>
 int main(){
 int a = 3;
 int b = 9;
 int sum;
 if(a==3 &&  b==9){
 sum = a+b;
 }
 printf("sum is %d",sum);
    return 0;
 }
Output

sum is 12

Both conditions are true in the above example program, so it executes the if block. In the if block we add a and b and stored the result in a sum variable.

Multiple if statements in C

To check more than one conditon we use multiple if statements

    
 #include<stdio.h>
 int main(){
 int age;
 printf("Enter Your age");
 scanf("%d",&age);
 if(age < 18){
    printf("Sorry you can't get license");
}
if(age > 16 && age < 18){
    printf("You should wait for one year to get a license");
}
if(age >= 18){
    printf("You can get a license");
}
return 0;
}
Output 1

Enter Your age18
You can get a license

Output 2

Enter Your age15
Sorry you can't get license

You can see that we asked the users for their age and we set some conditions. So it checks all the conditions and it executes the code where the condition is true.

What will happen if two conditions are true

    
 #include<stdio.h>
 int main(){
 int age;
 printf("Enter Your age");
 scanf("%d",&age);
 if(age < 18){
    printf("Sorry you can't get license\n");
}
if(age  >= 18){
    printf("You are age is just 18 so you can get a licence\n");
}
if(age < 19){
    printf("You can get a license\n");
}
return 0;
}
Output

Enter Your age18
You are age is just 18 so you can get a licence
You can get a license

Here both 2 and 3 if conditions are true so it executes both if block code in order.

We said that if the condition is false it skips the if block but we can let that code to executes in else block.
