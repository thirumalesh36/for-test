What is a break statement in C?

break statement is responsible to cut the flow of the program in a loop. When this break statement used in loops inside if statement the program will stop and come out of the loop. Generally it is used in loops and switch case statement.

You may understand this after loop concept so try to read this after reading those tutorials.

Syntax


break;

break statement example in while loop

    
#include <stdio.h>
int main()
{
int i =0;
while(i<=30)
{
printf("i = %d\n",i);

if (i==5)
{
break;
}

i++;
}
printf("i after break %d",i);

return 0;
}
Output

i = 0
i = 1
i = 2
i = 3
i = 4
i = 5
i after break 5

Here the while loop will continue up to i become 30 after that the loop will be terminated. But when i becomes 5 the loop is terminated because we use the break statement to terminate the loop after 5.

break statement in for loop

    
#include <stdio.h>
int main()
{
int i;
for(i = 0; i < 30; i++ ){

printf("i = %d\n",i);

if (i==6)
{
break;
}

}
printf("i after break %d",i);

return 0;
}
Output

i = 0
i = 1
i = 2
i = 3
i = 4
i = 5
i = 6
i after break 6

It just works as the sames as while loop when i becomes 6 the program will terminate from that and came out of the loop.

We will discuss how to use break statement in switch statement in the switch statement tutorial.
